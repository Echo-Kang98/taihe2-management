# 期现对冲策略

## 描述
设计并执行期现结合的对冲策略，为现货贸易头寸提供价格保护，同时保留价格有利变动的收益空间。

## 触发条件
- 用户说"有一批原油要采购，需要做对冲"
- 用户说"我们的库存需要保护吗"
- 价格剧烈波动时需要评估对冲需求

## 执行步骤
1. 识别现货敞口（待采购/已库存/已预售）
2. 评估对冲需求：
   - 已库存（已买入现货）→ 对冲价格下跌风险
   - 待采购（计划买入）→ 对冲价格上涨风险
3. 选择对冲工具：
   - 期货（SC原油、燃料油期货）
   - 期权（买入看涨/看跌保护）
   - 掉期（互换协议）
4. 确定对冲比例（部分对冲50-80% vs 全额对冲100%）
5. 确定对冲期限（与现货敞口匹配）
6. 测算对冲效果（最优/中性/最差情景）
7. 生成对冲方案并执行
8. 记录对冲台账，定期评估效果

## 输出格式
```
# 期现对冲策略方案
- 现货敞口：持有50万桶西非原油库存（采购均价80美元/桶）
- 敞口性质：已买入现货，多头敞口
- 对冲目标：保护价格下跌风险

## 敞口分析
| 情景 | 到期价格 | 现货盈亏 | 未对冲盈亏 |
|------|----------|----------|------------|
| 下跌至75 | -250万美元 | -250万×50万 | |
| 不变80 | 0 | 0 | |
| 上涨至85 | +250万美元 | +250万×50万 | |

## 对冲方案
### 方案A：期货完全对冲（100%）
- 操作：卖出50万桶SC原油期货（SC1）
- 锁定价格：约81美元/桶（考虑汇率和品质差）
- 效果：无论现货涨跌，净盈亏锁定在+50万美元（采购成本优势）

### 方案B：期权部分对冲（70%）
- 操作：买入35万桶看跌期权，执行价78美元/桶，权利金约1.5美元/桶
- 效果：保护下跌风险至77.5美元/桶，保留上涨收益至82美元/桶以上

### 方案C：50%期货+50%敞口
- 操作：卖出25万桶期货
- 效果：部分保护，下跌最多损失125万美元，上涨最多收益125万美元

## 情景对比
| 情景 | 方案A(期货100%) | 方案B(期权70%) | 方案C(50%+50%) |
|------|---------------|---------------|----------------|
| 价格跌至75 | +50万 ✅锁定 | +87.5万 ✅ | -125万 ❌ |
| 价格维持80 | +50万 ✅锁定 | -52.5万(权利金) | 0 |
| 价格涨至85 | +50万 ✅锁定(被套保) | 较大盈利 | +125万 |

## 推荐方案：方案A（期货100%对冲）
## 推荐理由：库存已采购，优先保护，放弃额外上涨收益换取确定性

## 对冲执行计划
- 执行时间：2026-03-31
- 对冲品种：SC原油期货 SC1
- 对冲数量：50万桶（500手）
- 执行价格：约605元/桶（约81美元/桶）
- 对冲期限：3个月
```

## 示例
输入：已采购50万桶西非原油，均价80美元/桶，需要对冲
输出：推荐100%期货对冲，锁定利润50万美元，避免价格下跌风险
