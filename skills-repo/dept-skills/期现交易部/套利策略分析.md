# 套利策略分析

## 描述
识别并分析跨品种、跨市场、跨合约的套利机会，包括原油-化工品产业链套利、跨期套利、跨市套利（SC/Brent/WTI）等。

## 触发条件
- 用户说"SC和Brent之间有套利机会吗"
- 用户说"原油和沥青的价差怎么样"
- 价差出现异常波动时自动触发分析

## 执行步骤
1. 采集相关品种价格数据：
   - 跨市：SC原油、Brent、WTI、Dubai原油
   - 跨期：SC原油近月-远月（SC1/SC2/SC3）
   - 产业链：原油-沥青、原油-PTA、原油-聚乙烯
2. 计算理论价差/比价
3. 对比历史价差分布（均值、标准差、置信区间）
4. 识别异常价差（超出±2σ）
5. 判断套利方向：
   - 正向市场（Contango）→ 买近卖远
   - 反向市场（Backwardation）→ 卖近买远
6. 计算预期收益和风险
7. 给出操作建议

## 输出格式
```
# 套利策略分析报告
- 分析时间：[datetime]

## SC-Brent 跨市套利
- SC主连价格：601元/桶（约USD 83/桶，汇率7.24）
- Brent主连价格：82.5美元/桶
- 价差（SC-Brent折美元）：83 - 82.5 = +0.5美元/桶
- 历史均值：-0.8美元/桶
- 当前分位：62%（略高于均值）
- 偏离程度：1.3美元/桶（超出均值+1σ）

## 套利分析
| 指标 | 当前值 | 历史均值 | 评估 |
|------|--------|----------|------|
| SC-Brent价差 | +$0.5 | -$0.8 | 略偏高 |
| 理论合理区间 | [-2.5, +1.5] | - | 正常 |

## 结论
当前SC-Brent价差略偏高但未达到显著套利窗口（需超过±2σ）
无明确跨市套利机会，建议观望

## SC跨期套利（近月-远月）
- SC1/SC2价差：-8元/桶（Contango结构）
- 年化展期成本：-8×12/1 = -96元/年/桶（约-1.6%）
- 理论合理展期成本：≈中美利差 ≈ 2%
- 当前展期成本低于利差，Contango偏窄

## 产业链套利：原油-沥青
- 原油价格：601元/桶
- 沥青期货价格：3850元/吨
- 出油率：约40%
- 理论成本：601÷0.4 = 1502.5元/吨
- 实际沥青-原油换算：3850元/吨
- 加工利润：3850 - 1502.5 = 2346元/吨（显著偏高）

⚠️ 沥青相对原油偏贵，建议：买入原油期货 + 卖出沥青期货（锁定加工利润）

## 操作建议
1. 原油-沥青产业链：可执行买SC原油空沥青策略，目标利润15元/吨
2. SC跨期：当前展期成本偏低，暂不推荐跨期套利
```

## 示例
输入：分析原油-沥青产业链套利机会
输出：沥青价格相对原油偏高，理论加工利润2346元/吨，建议买SC原油期货+空沥青期货，预期利润15元/吨
