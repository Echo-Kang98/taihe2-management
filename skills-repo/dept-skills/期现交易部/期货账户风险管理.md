# 期货账户风险管理

## 描述
实时监控期货账户的风险状况，包括保证金占用、浮动盈亏、风险度指标，确保不发生强平风险。

## 触发条件
- 用户说"查一下期货账户的风险情况"
- 每日收盘后自动触发风险报告
- 价格剧烈波动时（账户风险度>70%）自动预警

## 执行步骤
1. 获取期货账户实时数据：
   - 持仓明细（各合约、多空方向、持仓量）
   - 账户权益（起始权益、当日盈亏、浮动盈亏）
   - 保证金占用
   - 可用资金
2. 计算风险指标：
   - 风险度 = 保证金占用 / 账户权益 × 100%
   - 浮动盈亏
   - 当日盈亏
   - 盈亏比
3. 检查是否触发以下预警：
   - 风险度 > 50% ⚠️ 预警
   - 风险度 > 70% 🔴 警戒
   - 风险度 > 80% 🚨 强平预警
   - 单合约持仓超限
4. 模拟价格变动对账户的影响（压力测试）
5. 生成风险报告
