# 战略决策支持

## 描述
为高管提供石油/大宗商品市场战略分析，辅助重大投资决策和业务方向判断。

## 触发条件
- 高管询问"市场走势如何"、"我们应该怎么调整战略"
- 需要分析宏观市场事件对业务的影响
- 新项目立项前的战略评估

## 执行步骤
1. 收集近期原油/大宗商品价格数据（Brent、WTI、Dubai原油价格）
2. 分析主要产油国政策变化（OPEC+产量决议、地缘政治事件）
3. 结合公司现有持仓和贸易头寸，评估敞口风险
4. 生成《战略决策参考报告》，包含：市场快照、风险点、建议措施
5. 输出报告摘要，供高管快速决策

## 输出格式
```
## 市场战略分析报告
- 报告日期：[date]
- 原油价格区间：[range] 美元/桶
- 关键风险：[list]
- 敞口分析：[敞口品种、规模、方向]
- 战略建议：[1/2/3...]
- 置信度：[高/中/低]
```

## 示例
输入："分析当前原油市场，给出我们的战略建议"
输出：结合Brent 82美元/桶、OPEC+减产延续、库存下降等因素，建议短期内维持多头敞口，对冲策略建议...

## 依赖
- 财务经营部_资金状况查询
- 交易执行部_当前持仓查询
