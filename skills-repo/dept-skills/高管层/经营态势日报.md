# 经营态势日报

## 描述
每日汇总各业务部门关键经营数据，生成高管层经营态势日报，涵盖贸易量、利润、风险指标。

## 触发条件
- 用户说"生成今天的经营日报"
- 每日定时（早上8:00自动触发）
- 高管询问"今天经营情况怎么样"

## 执行步骤
1. 汇总昨日国际交易部、国内交易部、交易执行部的贸易成交数据
2. 获取财务经营部的资金和利润数据
3. 获取汇率经营部的外汇敞口和盈亏数据
4. 获取供应链运营部的库存和物流状态
5. 获取期现交易部的期货账户权益和持仓数据
6. 汇总生成《泰和嘉柏经营态势日报》

## 输出格式
```
# 泰和嘉柏经营态势日报
- 日期：[date]
- 报告人：[Agent name]

## 核心指标
| 指标 | 今日 | 昨日 | 环比 |
|------|------|------|------|
| 贸易总量 | xxx桶 | xxx桶 | +x% |
| 营业收入 | xxx万 | xxx万 | +x% |
| 净利润 | xxx万 | xxx万 | +x% |
| 外汇敞口 | xxx万 | xxx万 | +x% |

## 各部门经营摘要
[国际交易部] [国内交易部] [交易执行部] [汇率经营部] [供应链运营部] [期现交易部]

## 风险预警
- [warning items]

## 今日重点关注
- [priority items]
```

## 示例
输入：自动每日8:00触发
输出：完整日报，包含昨日原油贸易成交200万桶，实现利润1200万元，外汇敞口对冲比例达85%等数据

## 依赖
- 国际交易部_成交汇总
- 国内交易部_成交汇总
- 交易执行部_订单状态
- 财务经营部_利润快报
- 汇率经营部_敞口报告
- 供应链运营部_库存日报
- 期现交易部_期货账户权益
