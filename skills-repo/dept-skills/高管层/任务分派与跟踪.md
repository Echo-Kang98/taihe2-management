# 任务分派与跟踪

## 描述
接收高管指令，将跨部门任务分解并分派至对应部门Agent，全程跟踪执行进度并回告高管。

## 触发条件
- 高管说"把这个任务交给XX部门"
- 高管说"跟踪一下XX任务的进展"
- 高管说"帮我催一下XX部门的进度"

## 执行步骤
1. 解析高管指令，提取任务目标、涉及部门、截止时间
2. 判断任务类型（战略类/财务类/贸易类/物流类/合规类）
3. 调用尚书省流程，将子任务分发至各专业部门Agent
4. 生成任务跟踪看板，每隔2小时自动刷新进度
5. 完成后汇总各子任务结果，生成综合报告回告高管
6. 若超时或异常，推送预警消息

## 输出格式
```
## 任务分派单
- 任务ID：[JJC-YYYYMMDD-NNN]
- 主任务：[title]
- 发起人：[executive name]
- 创建时间：[timestamp]
## 子任务分解
| # | 子任务 | 承接部门 | 负责人 | 截止时间 | 状态 |
|---|--------|----------|--------|----------|------|
| 1 | ...    | ...      | ...    | ...      | 进行中 |
## 当前进度
[进度条] 3/5 子任务完成
## 阻塞项
- [blocker 1]
```

## 示例
输入："下季度原油采购计划需要各部门的配合，帮我分派下去"
输出：自动拆分为资金计划（财务经营部）、供应商谈判（国际交易部）、物流安排（供应链运营部）三个子任务，并建立跟踪看板
