# 国际供应商关系管理

## 描述
维护国际原油/化工品供应商信息，评估供应商绩效，辅助采购决策。

## 触发条件
- 用户说"查一下某供应商的资质和历史合作记录"
- 用户说"这个供应商的履约情况如何"
- 寻找新供应商时需要供应商比选

## 执行步骤
1. 查询供应商基础信息库（名称、国别、资质、联系方式）
2. 汇总与该供应商的历史交易记录（合同数量、履约情况、品质投诉）
3. 评估供应商绩效指标：
   - 按时交货率
   - 品质合格率
   - 价格竞争力
   - 配合度评分
4. 对比同类供应商的市场情况
5. 生成供应商评估报告

## 输出格式
```
# 供应商评估报告
- 供应商名称：[supplier name]
- 国别：[country]
- 合作开始时间：[date]

## 合作概况
- 累计合作合同：xx份
- 累计交易量：xxx万桶/公吨
- 合同金额合计：xxx万美元

## 绩效评分（满分10分）
| 指标 | 得分 | 说明 |
|------|------|------|
| 按时交货 | 9.0 | 近3笔均提前 |
| 品质达标 | 8.5 | API略低于合同均值 |
| 价格竞争力 | 8.0 | 略高于市场均价 |
| 配合响应 | 9.5 | 响应及时，配合度高 |

## 综合评分：8.8/10 ✅ 优质供应商

## 市场比选（如适用）
| 供应商 | 到岸价(CIF) | 品质 | 交期 |
|--------|-------------|------|------|
| 本供应商 | USD 82/桶 | API 33 | 30天 |
| 市场参考A | USD 81.5/桶 | API 34 | 28天 |
| 市场参考B | USD 83/桶 | API 32 | 35天 |
```

## 示例
输入：评估阿联酋Emirates Oil的合作绩效
输出：累计合作15份合同，综合评分8.9/10，品质稳定，建议继续合作

## 依赖
- 企业微信智能表格（供应商台账）
- 交易执行部_历史合同查询
