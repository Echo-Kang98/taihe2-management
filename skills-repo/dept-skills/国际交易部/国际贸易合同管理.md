# 国际贸易合同管理

## 描述
管理国际原油/化工品贸易合同的全生命周期，从条款审核、签署、生效、执行到关闭的全流程管理。

## 触发条件
- 用户说"有一份新合同需要审核"
- 用户说"查一下XX合同的状态"
- 合同快到期或执行完毕需要关闭

## 执行步骤
1. 接收合同文本（PDF/Word）或关键条款信息
2. 审核合同关键条款：
   - 价格条款（CIF/FOB/EXW）
   - 数量条款（溢短装允许范围）
   - 付款条款（T/T、信用证、银行保函）
   - 品质标准（API度、含硫量）
   - 争议解决条款
3. 标注合同风险点（不接受条款/需法务复核）
4. 在合同管理系统中创建合同档案
5. 设置执行里程碑提醒
6. 跟踪合同执行直至关闭

## 输出格式
```
# 国际贸易合同审核报告
- 合同编号：[contract no]
- 供应商/客户：[counterparty]
- 商品：[commodity] [quantity]
- 合同金额：[amount] USD/CNY
- 价格条款：[INCOTERMS 2020]

## 关键条款审核
| 条款 | 合同约定 | 风险评估 |
|------|----------|----------|
| 价格 | USD xx/桶 | ✅ 无异常 |
| 付款 | 信用证LC 90天 | ⚠️ 见备注 |
| 数量 | 50000桶 ±5% | ✅ 正常范围 |
| 品质 | API 32-36, S<1.5% | ✅ 符合标准 |

## 风险提示
- ⚠️ 付款条款中要求预付款30%，需评估供应商信用
- ⚠️ 争议解决适用英国法律，建议增加仲裁条款

## 审核结论：[通过/条件通过/驳回]
## 法务复核：[是/否]
```

## 示例
输入：审核一份30万桶俄罗斯ESPO原油进口合同，CIF南京
输出：条款基本符合公司标准，付款方式为T/T30%+L/C70%可接受，建议增加品质仲裁条款，最终通过审核
