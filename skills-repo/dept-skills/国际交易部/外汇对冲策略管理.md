# 外汇对冲策略管理

## 描述
针对国际贸易中的外汇敞口，制定并执行外汇对冲策略，锁定成本/收益，规避汇率波动风险。

## 触发条件
- 用户说"这笔美元应付账款需要做对冲"
- 用户说"我们的外汇敞口有多大"
- 汇率波动超过±2%时自动触发对冲建议

## 执行步骤
1. 识别外汇敞口（美元应收/应付、欧元结算等）
2. 获取当前外汇敞口汇总（金额、币种、到期日）
3. 查询当前远期汇率和期权价格
4. 结合敞口金额和风险偏好，制定对冲方案：
   - 方案A：远期结售汇（锁定汇率）
   - 方案B：外汇期权（保留上行空间）
   - 方案C：部分对冲（50%锁汇+50%敞口）
5. 模拟测算不同方案下的盈亏平衡点
6. 协助执行对冲交易
7. 记录对冲台账，定期评估有效性

## 输出格式
```
# 外汇对冲策略建议
- 敞口编号：[exposure no]
- 敞口类型：[应收美元/应付欧元/...]
- 敞口金额：[amount] USD
- 到期日：[maturity]
- 当前即期汇率：CNY/USD 7.25

## 对冲方案比较
| 方案 | 工具 | 对冲比例 | 锁定汇率 | 优点 | 缺点 |
|------|------|----------|----------|------|------|
| A | 远期 | 100% | 7.23 | 确定性强 | 无上行空间 |
| B | 期权 | 80% | 7.20 | 保留上行 | 权利金成本 |
| C | 远期+期权 | 50%+30% | 7.22 | 平衡 | 结构复杂 |

## 推荐方案：方案A
## 推荐理由：[rationale]
## 对冲后效果：[calculated result]
```

## 示例
输入：对冲下月5000万美元应付货款
输出：建议100%远期锁汇，锁定汇率7.23，结汇成本确定，可完全规避人民币贬值风险
