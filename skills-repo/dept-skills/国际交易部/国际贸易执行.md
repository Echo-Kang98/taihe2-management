# 国际贸易执行

## 描述
执行国际原油/化工品贸易全流程，从合同签订、货权转移、结算到交付，全程跟踪管理。

## 触发条件
- 国际贸易团队说"有一笔新进口原油合同需要执行"
- 用户说"查一下XX进口合同的执行进度"
- 货物到港前需要协调清关和物流

## 执行步骤
1. 接收贸易合同信息（供应商、商品、数量、价格、装运期、结算条款）
2. 确认CIQ检验、报关资料准备
3. 跟踪装运状态（与供应商/船代保持联络）
4. 货物到港后，协调清关、仓储
5. 计算溢短装和品质波动，确认最终结算量
6. 准备结算单据（发票、提单、CIQ证书、原产地证）
7. 提交财务部门进行外汇结算
8. 更新合同执行台账

## 输出格式
```
# 国际贸易执行跟踪
- 合同号：[contract no]
- 供应商：[supplier name]
- 商品：[commodity] - [quantity]桶/公吨
- 合同金额：[amount] USD/CNY
- 装运期：[loading period]
- 装港：[load port]
- 卸港：[discharge port]

## 执行节点
| 节点 | 计划时间 | 实际时间 | 状态 |
|------|----------|----------|------|
| 合同签订 | 2026-03-01 | 2026-03-01 | ✅完成 |
| 装运确认 | 2026-03-15 | 进行中 | 🔄 |
| 到港清关 | 2026-04-01 | 待执行 | ⏳ |
| 仓储入库 | 2026-04-03 | 待执行 | ⏳ |
| 结算付款 | 2026-04-15 | 待执行 | ⏳ |

## 当前状态：[执行中/已完成/异常]
## 待办事项：[list]
```

## 示例
输入：执行50万桶西非原油进口合同，FOB尼日利亚，4月到港
输出：已完成合同录入，装运期为3月15-20日，已协调CIQ检验，预计4月2日到港，已安排南京龙潭仓储库

## 依赖
- 国际交易部_合同条款审核
- 汇率经营部_外汇结算
- 供应链运营部_仓储协调
- 交易执行部_物流跟踪
