# 套利窗口监测

## 描述
监测跨市场、跨币种的外汇套利机会，包括境内外汇差（CNY/CNH）、远期升贴水异常、交叉汇率偏离等。

## 触发条件
- 用户说"现在有什么套利机会"
- 每日定时监测（早上9:30开盘后）
- 汇率出现异常波动时

## 执行步骤
1. 采集境内外汇率数据：
   - USD/CNH（离岸）、USD/CNY（在岸）即期汇率
   - 境内外汇差点数
   - 远期升贴水（1周/1月/3月/6月/1年）
   - EUR/USD、USD/JPY等交叉汇率
2. 计算套利空间：
   - 境内外汇差 vs 套利成本（点差、滑点、资金成本）
   - 远期升贴水年化 vs 利差
   - 交叉汇率 vs 三角套利理论值
3. 评估机会的可行性（资金规模、时效性、对手方）
4. 生成套利机会报告

## 输出格式
```
# 套利窗口监测报告
- 监测时间：[datetime]
- USD/CNY 在岸即期：7.2510
- USD/CNH 离岸即期：7.2545
- 境内外汇差：35点（CNY<CNH，即在岸相对便宜）

## 境内外套利监测
| 指标 | 当前值 | 套利成本 | 空间 | 窗口状态 |
|------|--------|----------|------|----------|
| USD/CNY 汇差 | +35pips | 约20pips | 15pips | ⚠️ 收窄 |
| 3月远期升水年化 | 1.8% | 中美利差-2.0% | 0.2% | ⚠️ 轻微 |

## 交叉汇率套利
- EUR/USD 即期：1.0850
- EUR/CNY 即期：8.4970
- USD/CNY 即期：7.2510
- 理论EUR/CNY：1.0850×7.2510 = 7.8674
- 实际EUR/CNY：8.4970
- 偏差：+6296pips（偏离过大，⚠️注意是否为数据异常）

## 远期结构机会
- 6月远期USD/CNY：7.1950（年化贬值1.9%）
- 若买入同期限USD利率5.2%，CNY利率3.3%，利差1.9%
- 远期价格合理，无明显套利机会

## 综合结论
当前无显著套利窗口。境内外汇差收窄至15pips，不足以覆盖资金成本。
```

## 示例
输入：监测今日套利窗口
输出：境内外汇差35点，收窄至临界，建议关注离岸资金面变化；远期贴水年化1.8%接近利差，无明显套利空间
