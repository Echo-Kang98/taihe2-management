# 汇率走势分析

## 描述
分析人民币/美元、欧元等主要货币汇率走势，研判影响因素，预测短期和中长期汇率方向，为外汇决策提供支持。

## 触发条件
- 用户说"美元/人民币接下来怎么走"
- 月度汇率分析会议前
- 重大宏观事件（美联储加息/中国GDP发布）后

## 执行步骤
1. 收集近期汇率数据（1个月/3个月/1年期走势）
2. 分析核心驱动因素：
   - 美元指数（DXY）走势
   - 中美利差（10年期国债收益率差）
   - 贸易顺差/逆差数据
   - 外汇储备变动
   - 央行政策表态
3. 技术面分析（支撑位/阻力位）
4. 综合研判短期（1-2周）和中期（1-3月）汇率方向
5. 生成汇率分析报告

## 输出格式
```
# 汇率走势分析报告
- 分析日期：[date]
- 分析周期：短期（1-2周）/ 中期（1-3月）

## 汇率现状
- USD/CNY 即期：7.2510
- USD/CNY 1年远期：7.1950
- 隐含贬值预期：约0.77%（年化）

## 主要驱动因素分析
| 因素 | 当前状态 | 影响方向 | 权重 |
|------|----------|----------|------|
| 美元指数 | 104.5，反弹中 | USD↑ | 30% |
| 中美利差 | -150BP（中国低） | CNY↓ | 35% |
| 贸易顺差 | 580亿美元（偏低） | CNY↓ | 20% |
| 外资流入 | 净流入减少 | CNY↓ | 15% |

## 技术面
- 支撑位：7.20 / 7.18
- 阻力位：7.28 / 7.32
- 趋势判断：震荡偏弱

## 综合研判
- 短期（1-2周）：7.23-7.30 区间震荡，倾向测试7.28
- 中期（1-3月）：破7.30概率55%，目标7.35

## 贸易建议
✅ 结汇窗口：7.28以上可加大结汇比例
⚠️ 购汇时机：暂缓，等待回调至7.20以下
```

## 示例
输入：分析美元/人民币未来一个月走势
输出：预计短期维持7.23-7.30震荡，4月前若中国出口数据不及预期，美元可能反弹至7.32，建议出口收汇在7.28以上分批结汇
