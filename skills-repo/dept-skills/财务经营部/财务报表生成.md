# 财务报表生成

## 描述
自动汇总业务数据，生成符合会计准则的财务报表（P&L、资产负债表、现金流量表），支持月度/季度/年报格式。

## 触发条件
- 财务人员说"生成月度财务报表"
- 高管要求"看看本月P&L"
- 审计/监管需要的定期报告

## 执行步骤
1. 从各业务系统拉取本月贸易成交记录（国际+国内+期货）
2. 获取采购成本、销售收入、仓储物流费用等成本数据
3. 获取外汇兑换记录和汇兑损益数据
4. 获取资金利息收支数据
5. 按会计准则计算：营业收入、营业成本、毛利、费用、净利润
6. 生成三表（利润表、资产负债表、现金流量表）
7. 输出PDF格式财务报表

## 输出格式
```
# 泰和嘉柏财务报表
- 期间：[月份/季度/年度]
- 编制日期：[date]
- 货币单位：人民币万元

## 利润表（P&L）
- 营业收入：xxx
- 其中：原油贸易 xxx / 化工品 xxx / 其他 xxx
- 营业成本：xxx
- 毛利：xxx（毛利率 xx%）
- 期间费用：xxx（管理/财务/销售）
- 汇兑损益：xxx
- 净利润：xxx

## 资产负债表（摘要）
- 流动资产合计：xxx
- 应收账款：xxx
- 存货：xxx
- 流动负债合计：xxx
- 应付账款：xxx

## 现金流量表（摘要）
- 经营活动现金流：xxx
- 投资活动现金流：xxx
- 筹资活动现金流：xxx
```

## 示例
输入：生成2026年1月财务报表
输出：1月营收3.2亿元，净利润2800万元，毛利率18.2%，应收账款周转天数28天

## 依赖
- 交易执行部_订单成交明细
- 财务经营部_资金流水
- 汇率经营部_汇兑损益台账
