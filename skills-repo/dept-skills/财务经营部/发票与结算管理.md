# 发票与结算管理

## 描述
管理公司发票开具、应收应付账款结算，跟踪结算进度，确保资金及时回笼。

## 触发条件
- 用户说"查一下XX客户的应收账款"
- 销售团队通知货物已发出，需要开具发票
- 供应商来票，需要核对付款

## 执行步骤
1. 接收业务单据（合同号、货物数量、单价、结算条件）
2. 计算发票金额（含税/不含税）
3. 在财务系统中创建应收/应付记录
4. 设置到期付款提醒
5. 跟踪付款状态，更新台账
6. 对超期30天以上的款项发起催收流程

## 输出格式
```
# 发票结算台账
- 发票号：[invoice no]
- 开票日期：[date]
- 客户/供应商：[name]
- 金额：[amount]（含税）
- 税率：13%
- 到期日：[due date]
- 当前状态：[未到期/已付款/逾期x天]
- 逾期利息：[if applicable]

## 汇总
| 类型 | 笔数 | 总额 | 已结清 | 待结算 | 逾期 |
|------|------|------|--------|--------|------|
| 应收账款 | xx | xxx万 | xxx万 | xxx万 | xxx万 |
| 应付账款 | xx | xxx万 | xxx万 | xxx万 | xxx万 |
```

## 示例
输入：录入新一笔原油销售发票，客户新加坡ABC Oil，金额1200万美元，账期90天
输出：发票已录入，到期日为2026年7月1日，已设置付款提醒

## 依赖
- 交易执行部_合同执行状态
- 财务经营部_结算台账写入
- 企业微信待办（付款提醒）
