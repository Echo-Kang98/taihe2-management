# 仓储管理

## 描述
管理公司租用的原油/化工品仓储设施，协调出入库操作，监控库存安全，确保仓储资产高效运转。

## 触发条件
- 用户说"查一下XX仓库的库存情况"
- 货物需要入库或出库时
- 仓储成本结算时

## 执行步骤
1. 查询各仓库的可用库容（总容量、已用容量、可用容量）
2. 接收出入库指令：
   - 入库：核对货权转移文件、计量入库数量、品质检验
   - 出库：核对提货指令、计量出库数量、确认货权释放
3. 协调仓储公司执行操作
4. 实时更新库存台账（手动输入或对接WMS系统）
5. 监控罐区状态（温度、压力、液位）
6. 月末与仓储公司对账，确认仓储费用
7. 识别异常情况（爆仓风险、货权纠纷、品质异常）
