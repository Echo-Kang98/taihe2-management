# 合规检查审计

## 描述
对公司业务流程进行合规检查，确保业务操作符合法律法规、公司制度和行业规范，识别合规风险并推动整改。

## 触发条件
- 用户说"做一次合规检查"
- 季度/年度内部审计
- 监管政策变化需要自查时

## 执行步骤
1. 确定合规检查范围和重点领域：
   - 贸易合规（出口管制、贸易制裁）
   - 金融合规（外汇管理、跨境支付）
   - 税务合规（转让定价、关税申报）
   - 数据合规（客户信息保护）
2. 收集业务记录（合同、单据、审批流程）
3. 逐项核对合规要点：
   - 交易对手是否在制裁名单（OFAC/UN/EU）
   - 外汇收支是否超过额度
   - 单据是否真实完整
   - 审批流程是否完整
4. 识别违规项（严重/一般/观察项）
5. 评估风险等级
6. 提出整改建议
7. 跟踪整改进度直至关闭
