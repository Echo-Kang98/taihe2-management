# 物流协调管理

## 描述
协调石油/化工品运输全流程，管理海运、陆运、仓储等物流资源，优化运输成本和时效。

## 触发条件
- 用户说"有一批货需要安排运输"
- 用户说"查一下这批货到哪了"
- 货物到港/到库前需要协调物流安排

## 执行步骤
1. 获取货物信息（数量、体积、品类、起运地、目的地）
2. 评估运输方式（海运整车/铁路/管道/海+陆联运）
3. 向物流供应商询价（3家以上比价）
4. 选择最优方案（综合成本+时效+安全性）
5. 安排运输计划：下达运单、跟踪在途
6. 货物到达后确认卸货状态，处理货损
7. 更新物流台账，统计运费成本

## 输出格式
```
# 物流执行报告
- 运单号：[waybill no]
- 货物：[commodity] [quantity]
- 起运地：[origin]
- 目的地：[destination]
- 运输方式：[mode]

## 物流方案比选
| 供应商 | 报价 | 预计时效 | 方案 |
|--------|------|----------|------|
| 中远海运 | 85万 | 12天 | 海运直达 |
| 招商物流 | 82万+5万港建 | 14天 | 海+江 |
| 中铁物流 | 95万 | 8天 | 铁路直达 |

## 选定方案：招商物流（成本最优）
## 运费总计：87万元

## 在途跟踪
| 日期 | 位置 | 状态 | 备注 |
|------|------|------|------|
| 2026-03-01 | 南京港 | 已装船 | - |
| 2026-03-03 | 长江口 | 航行中 | 正常 |
| 2026-03-10 | 目的港 | 待卸货 | - |

## 运费统计
- 本月累计运费：xxx万元
- 较上月同期：+x%（受油价上涨影响）
```

## 示例
输入：安排50万桶ESPO原油从宁波港到南京龙潭库的陆运
输出：比选3家物流商，选择海进江方案，预计运费68元/吨，总运费34万元，预计3月10日到港

## 依赖
- 供应链运营部_仓储库存查询
- 交易执行部_订单确认
- 企业微信智能表格（物流台账）
