# 单据与文件管理

## 描述
管理石油/大宗商品贸易全流程所需单据，包括提单(B/L)、原产地证(CO)、商检证书(CIQ)、发票(INV)、装箱单(PL)等，确保单证一致、交付及时。

## 触发条件
- 用户说"这批货的提单准备好了吗"
- 用户说"需要向银行提交什么单据"
- 信用证结算需要审核单证一致

## 执行步骤
1. 确认结算方式（T/T、LC、信用证）
2. 根据结算方式确定所需单据清单
3. 各部门协同准备单据：
   - 仓储：准备仓单或货权转移证明
   - 商检：CIQ检验证书
   - 贸易：发票、装箱单、合同
   - 物流：提单（B/L）、海运单（SWB）
4. 审核单据一致性（单证相符、单单相符）
5. 向银行/客户提交单据
6. 归档已执行单据，建立电子档案

## 输出格式
```
# 单据审核报告
- 合同号：[contract no]
- 发票号：[invoice no]
- 结算方式：T/T 30天 / 不可撤销信用证

## 单据清单
| 单据名称 | 份数 | 准备方 | 状态 | 完成时间 |
|----------|------|--------|------|----------|
| 商业发票(Commercial Invoice) | 3 | 国际交易部 | ✅已备 | 2026-03-15 |
| 装箱单(Packing List) | 3 | 仓储部 | ✅已备 | 2026-03-15 |
| 提单(Bill of Lading) | 3 | 船代/物流 | 🔄准备中 | 预计3/16 |
| 原产地证(COO) | 2 | 单证 | ⏳待启动 | - |
| CIQ检验证书 | 1 | 商检机构 | ⏳待出证 | 3/18 |

## 单证一致性审核
- 发票金额 vs 合同金额：✅相符
- 提单数量 vs CIQ数量：✅相符
- 船名航次 vs 装港记录：✅相符

## 提交计划
- 向银行提交：2026-03-19
- 预计收汇：2026-04-18（T/T 30天）

## 状态：[准备中/已提交/已完成]
```

## 示例
输入：50万桶西非原油LC结算单据审核
输出：全套单据已准备完毕，商检证书显示API 33.2（符合合同32-36标准），提单数量499800桶（±5%范围内），单证审核通过，已提交星展银行议付
