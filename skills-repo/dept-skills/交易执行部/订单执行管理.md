# 订单执行管理

## 描述
管理石油/化工品贸易订单的全生命周期，从订单录入、生效、执行、变更到关闭的全流程管理。

## 触发条件
- 用户说"有一笔新订单需要录入"
- 用户说"查一下XX订单的执行状态"
- 订单执行过程中需要变更（数量/价格/交割期）

## 执行步骤
1. 接收订单信息（合同号、商品、数量、价格、客户/供应商）
2. 在订单管理系统中创建订单记录
3. 触发执行流程：
   - 通知仓储物流团队备货/提货
   - 通知财务部门准备收付款
   - 通知质量检验团队安排CIQ/质检
4. 跟踪执行节点（已确认→已备货→已发货→已到货→已完成）
5. 处理执行异常（延迟、质量问题、数量差异）
6. 订单完成后更新台账并关闭订单

## 示例
输入：录入新销售订单，向浙江能源销售5000吨柴油，单价4800元/吨
