# 船务与航次管理

## 描述
管理国际原油/大宗商品运输的船务安排，包括租船、跟踪航次、计算运费、处理海运文件。

## 触发条件
- 用户说"有批原油需要租船"
- 用户说"查一下这艘船现在到哪了"
- 货物装港/卸港需要协调船代

## 执行步骤
1. 获取货物信息（数量、装港、卸港、受载期）
2. 评估船型需求（VLCC/Aframax/Panamax/MR）
3. 向船东/租船经纪人询价（程租/期租）
4. 选择最优方案（运费费率、交货期、船况评级）
5. 签订租船合同（制定航次租船合同VOYAGECHART）
6. 跟踪航次状态：
   - 装港准备就绪（NOR）
   - 装货进度
   - 离港动态
   - 航行中
   - 卸港到港
7. 计算运费（根据装卸港数量、LAYTIME计算）
8. 处理海运文件（提单B/L、海运单SWB）

## 输出格式
```
# 航次执行跟踪
- 航次编号：[voyage no]
- 船名：[vessel name]
- 船型：VLCC（30万吨级）
- 租船合同：程租合同
- 货物：[commodity] [quantity]
- 装港：[load port]
- 卸港：[discharge port]
- 运费费率：WS 75（即期市场）

## 航次时间线
| 节点 | 预计时间 | 实际时间 | 状态 |
|------|----------|----------|------|
| 受载期 | 2026-03-15~20 | - | ⏳ |
| 装港NOR | - | - | ⏳ |
| 装货完成 | - | - | ⏳ |
| 离港 | - | - | ⏳ |
| 卸港到港 | 2026-04-05 | - | ⏳ |

## 运费估算
- 运费总额：约USD 3,200,000（按WS75和运距计算）
- 已付运费：[amount] USD
- 滞期费（如有）：[amount] USD

## 当前动态：[vessel name] 预计3月16日抵港
```

## 示例
输入：安排30万吨西非原油租船，从安哥拉LOBITO港到中国宁波港
输出：已租Maran Artemis轮，运费WS78，受载期3月20-25日，预计4月8日到宁波港
