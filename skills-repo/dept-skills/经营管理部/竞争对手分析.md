# 竞争对手分析

## 描述
持续追踪石油/大宗商品贸易行业主要竞争对手的战略动向、市场份额、产品结构、价格策略等，支持公司战略决策。

## 触发条件
- 用户说"我们的主要竞争对手最近有什么动向"
- 季度竞争对手分析
- 制定市场策略前

## 执行步骤
1. 确定竞争对手范围：
   - 直接竞争对手（同类民营贸易商）
   - 间接竞争对手（国营/外资贸易商）
   - 潜在进入者
2. 收集竞争对手信息：
   - 公开信息（年报、新闻、公告）
   - 行业协会数据
   - 行业研究机构报告
   - 市场传言（需核实）
3. 分析维度：
   - 市场规模和市场份额变化
   - 业务结构和盈利模式
   - 客户结构和渠道布局
   - 价格策略和竞争手段
   - 战略动向和新业务拓展
4. 对比公司与竞争对手的优劣势
5. 生成竞争对手分析报告

```
# 竞争对手分析报告

- 更新日期：[datetime]

|----------|------|-------------|----------|----------|
| 振华石油 | 央企背景 | 8% | 资金成本低、货源稳定 | 决策慢、机制僵 |

## 重点竞争对手深度分析：振华石油
### 基本情况
- 背景：央企中国兵工集团旗下
- 年营收：约150亿元
- 主营：原油/化工品进出口

### 最新动向（Q1）
- 🔴 扩张动作：在新加坡设立新贸易平台，目标辐射东南亚市场
- 🟡 融资动作：获批30亿美元银行授信，资金实力大增
- 🟢 合作动态：与伊拉克国油续签年度供油协议（200万桶/月）

### 竞争影响
- 威胁评估：对我们国际原油业务的直接竞争加剧
- 机会：其重心在国际市场，国内化工品市场仍有空间

## 市场份额变化趋势
- 2024年：振华8%→2025年8.2%→2026年Q1 8%（持平）
- 2024年：泰和嘉柏1.8%→2025年2%→2026年Q1 2.1%（小幅增长）

## 竞争策略建议
### 针对振华石油
- 差异化：发挥民企灵活性，在化工品小品种、紧急订单响应上建立优势
- 客户分层：聚焦中等规模炼厂（振华不重视的长尾客户）

### 针对托克
- 学习：借鉴其全球期现结合对冲模式
- 回避：避免在跨市套利等成熟业务上正面竞争

## SWOT对比（vs振华石油）
| | 泰和嘉柏 | 振华石油 |
|--|---------|---------|
| 优势(S) | 决策快、风控强、客户服务灵活 | 资金成本低、货源稳定、规模大 |
| 劣势(W) | 资金成本高、货源依赖进口 | 决策慢、客户响应慢 |
| 机会(O) | 化工品市场增长、新能源转型服务 | 一带一路政策支持 |
| 威胁(T) | 央企挤压、市场价格透明 | 央企整合行业 |

## 结论
振华石油Q1扩张信号明显，但主要精力在国际市场。建议Q2加大国内化工品市场投入，避免正面竞争。
```
