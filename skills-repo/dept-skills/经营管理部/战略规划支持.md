# 战略规划支持

## 描述
辅助高管制定中长期战略规划，包括市场拓展、新业务布局、竞争对手分析等战略议题的研究支持。

## 触发条件
- 高管说"我们需要规划一下未来3年的战略方向"
- 董事会战略会议前需要研究材料
- 进入新市场/新业务领域前需要可行性研究

## 执行步骤
1. 接收战略研究课题（范围、目标、时间节点）
2. 进行宏观环境分析（PEST：政治/经济/社会/技术）
3. 进行行业分析：
   - 全球/中国石油化工市场供需格局
   - 主要竞争格局（国际大石油公司/国内三桶油/民营贸易商）
   - 行业发展趋势（新能源替代、碳中和、物流优化）
4. 进行竞争情报分析：
   - 主要竞争对手的战略动向
   - 竞争优势/劣势对比（SWOT）
5. 进行内部能力评估：
   - 资金实力、风控能力、客户资源、人才队伍
6. 提出战略选项和推荐方案
7. 生成《战略规划研究报告》

## 输出格式
```
# 战略规划研究报告
- 课题：[战略课题名称]
- 研究周期：[date range]
- 报告级别：[机密/内部]

## 宏观环境分析（PEST）
- 政治(P)：一带一路政策支持民营贸易；RCEP降低关税壁垒
- 经济(E)：全球原油需求预计2030年达峰；中国化工品需求稳健
- 社会(S)：双碳目标推动能源转型；化工品消费升级
- 技术(T)：区块链溯源应用；期货期权工具丰富

## 行业分析
### 市场供需
- 全球原油需求：1.02亿桶/天（2026），年增速1.8%
- 中国原油进口依存度：73%（2026），维持高位
- 化工品市场：PTA、聚烯烃维持紧平衡

### 竞争格局
| 参与者 | 市场份额 | 优势 | 劣势 |
|--------|----------|------|------|
| 中石油/中石化 | 40% | 资源、渠道 | 灵活性差 |
| 振华石油 | 8% | 国际化 | 体量有限 |
| 泰和嘉柏 | 2% | 灵活、风控 | 资金成本 |
| 托克/维多 | 15% | 全球网络 | 国内渠道弱 |

## SWOT分析
| | 有利 | 不利 |
|--|------|------|
| 内部 | 决策灵活、风控严格、客户资源 | 资金成本高、人才储备 |
| 外部 | 市场需求稳健、政策支持 | 大宗商品价格波动、竞争加剧 |

## 战略选项
### 方案A：深耕国际原油贸易
- 定位：成为中国领先的原油进口贸易商
- 策略：扩大国际供应商合作、增加VLCC运力锁定
- 目标：3年内原油贸易量达1000万桶/月

### 方案B：拓展化工品产业链
- 定位：从原油向化工品产业链延伸
- 策略：与国内大型炼厂深度合作、拓展PTA/聚烯烃贸易
- 目标：3年内化工品占比达30%

### 方案C：期现结合专业化
- 定位：打造大宗商品衍生品对冲专业能力
- 策略：扩充期货团队、建立量化交易系统
- 目标：期现对冲年收益5000万+

## 推荐方案：方案B（拓展化工品产业链）
## 推荐理由：[rationale]

## 实施路径
1. 2026年：建立化工品团队，试水PTA贸易（月5000吨）
2. 2027年：扩大化工品种类，月贸易量达2万吨
3. 2028年：化工品占比达30%，形成新的利润增长点
```

## 示例
输入：研究未来3年战略方向
输出：推荐方案B（化工品产业链拓展），预计2028年化工品占比达30%，年新增利润2000万+
