# 国内市场情报收集

## 描述
实时监控国内石油/化工品市场动态，收集价格信息、竞品动态、政策变化，为国内贸易决策提供情报支持。

## 触发条件
- 用户说"今天国内市场有什么动静"
- 用户说"收集一下主要炼厂的最新报价"
- 价格大幅波动时自动预警

## 执行步骤
1. 采集主要信息源：
   - 卓创资讯/隆众资讯价格数据
   - 国家发改委油价调整通知
   - 主要炼厂（中石油/中石化/民营炼厂）报价
   - 期货市场行情（INE原油/沥青期货）
2. 识别关键信息变化：
   - 价格涨跌（标记涨跌超2%的品种）
   - 新产能投放/炼厂检修信息
   - 政策通知（出口配额、环保政策等）
3. 生成每日市场情报摘要
4. 对异常波动发送即时预警

## 输出格式
```
# 国内市场情报日报
- 日期：[date]
- 采集时间：[time]

## 重点品种行情
| 品种 | 今日均价 | 昨日均价 | 涨跌 | 涨跌幅 |
|------|----------|----------|------|--------|
| 华东汽油 | 7850 | 7800 | +50 | +0.64% |
| 华东柴油 | 4820 | 4780 | +40 | +0.84% |
| 航煤 | 5100 | 5150 | -50 | -0.97% |

## 重要情报
🔴 [原油] 发改委明日下调零售限价，预计汽油降150元/吨
🟡 [炼厂] 镇海炼化明日开始检修，为期15天，影响柴油供应
🟢 [政策] 第二批出口配额下发，柴油出口量增加50万吨

## 竞品动态
- 中石油华东公司：今日出货量2000吨，库存偏低
- 某民营炼厂：报价略低于市场，有促销意愿

## 风险提示
⚠️ 柴油跨区套利窗口收窄，建议加快出货节奏
```

## 示例
输入：今日市场情报收集
输出：华东柴油均价4820元/吨，较昨日上涨40元，镇海炼化检修导致供应偏紧，预测短期仍有上涨空间

## 依赖
- 外部数据接口（卓创/隆众资讯，如有）
- 交易执行部_仓储库存数据
