# 区域市场分析

## 描述
分区域分析国内石油/化工品市场供需状况、价格走势，为国内贸易决策提供市场情报支持。

## 触发条件
- 用户说"华东地区的柴油市场怎么样"
- 用户说"这个月国内汽柴油走势如何"
- 制定月度国内贸易策略前

## 执行步骤
1. 收集各区域市场数据（华东/华南/华北/华中/西南）
2. 汇总主要市场成交价格（汽油/柴油/航煤/化工品）
3. 分析影响因素：
   - 炼厂开工率
   - 库存水平
   - 下游需求（工地/物流/农业）
   - 进口资源到港情况
4. 对比各区域价差（跨区套利机会）
5. 预测未来1-2周价格走势
6. 生成区域市场分析报告

## 输出格式
```
# 国内区域市场分析报告
- 分析周期：[date range]
- 覆盖区域：[华东/华南/华北/华中/西南]

## 汽油市场
- 华东均价：xxxx元/吨（环比 +x%）
- 主要影响因素：[list]
- 后市预测：[看涨/盘整/回落]

## 柴油市场
- 华东均价：xxxx元/吨（环比 +x%）
- 库存水平：中等偏低
- 跨区套利窗口：[开启/关闭]
  - 华东→西南：xxx元/吨运费，价差不足以覆盖运费

## 各区域价差矩阵
| | 华东 | 华南 | 华北 |
|--|------|------|------|
| 汽油 | 基准 | -30 | +20 |
| 柴油 | 基准 | -50 | +10 |

## 贸易机会
1. [opportunity 1]
2. [opportunity 2]

## 操作建议：[买入/卖出/观望]
```

## 示例
输入：分析华东柴油市场现状
输出：华东柴油均价4820元/吨，库存偏低，预测未来一周上涨50-80元，建议适当增加采购头寸

## 依赖
- 交易执行部_市场情报数据
- 企业微信智能表格（市场数据台账）
