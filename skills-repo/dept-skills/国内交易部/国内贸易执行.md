# 国内贸易执行

## 描述
执行国内原油/化工品贸易全流程，协调国内炼厂、贸易商、仓储物流等资源，完成购销执行。

## 触发条件
- 国内贸易团队说"有一批国产原油需要安排销售"
- 用户说"查一下华东区域的市场情况"
- 国内采购/销售合同需要快速响应

## 执行步骤
1. 接收国内贸易需求（采购或销售方向）
2. 扫描国内市场资源池（国内炼厂报价、贸易商报价）
3. 匹配最优资源：考虑价格、物流成本、交割便利性
4. 发起询价/报价流程
5. 确认成交后签订国内贸易合同
6. 协调仓储/物流执行
7. 跟踪货权转移和结算
8. 更新国内贸易执行台账

## 输出格式
```
# 国内贸易执行报告
- 执行日期：[date]
- 方向：[采购/销售]
- 商品：[commodity] - [quantity]公吨/桶
- 成交价：[price] 元/公吨（含税）
- 合同金额：[total amount] 万元
- 客户/供应商：[counterparty]
- 交割方式：[仓库自提/送货/标准仓单]

## 市场扫描
| 来源 | 报价 | 品质 | 到货期 | 综合得分 |
|------|------|------|--------|----------|
| 燕山石化 | 4200 | 国标 | 3天 | 9.0 |
| 齐鲁石化 | 4180 | 国标 | 5天 | 8.5 |
| 市场参考 | 4150 | 国标 | 7天 | 7.8 |

## 选取引用：燕山石化
## 执行状态：[已完成/进行中/异常]
```

## 示例
输入：紧急采购2000吨柴油，3天内到货
输出：匹配到燕山石化和中化泉州均有现货，协调燕山石化最快2天到货，已确认执行
